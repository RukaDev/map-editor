# Array

## Functions

### `removeFromArray`

Removes a specific value from an array.

```js
Array.removeFromArray(arr: array, value: any) -> none
```

### `removeFromArrayAll`

Removes all occurances of a specific value from an array.

```js
Array.removeFromArrayAll(arr: array, value: any) -> none
```
